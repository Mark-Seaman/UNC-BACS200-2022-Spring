### Using Version Control
* Pull Code - bring changes to your computer
* Make Changes - edit code on your computer
* Commit files to repo
* Push Code - send changes to Github


### Building Web Pages
* Edit files locally
* Test files locally
* Commit files to repo
* Test files remotely
